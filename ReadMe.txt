Library Management System 
 
Автор: Стрільцова Аліна Олегівна       
Група ІС-62, II курс    
Номер заліковки ІС6223 
     
Для перегляду "Електронної бібліотеки" потрібно натиснути кнопку "START".
Перед вами відкриється форма з відповідними даними, а саме увесь список наданих книг, 
інформація про обрану книгу, поле для пошуку, а також поле для фільтрації.
Додаток надає змогу:
 *додавати("Додати"), 
 *видаляти("Видалити"), 
 *редагувати("Редагувати"),
 *зберігати("Зберегти")
 *оновлювати("Оновити") дані,
 *здійснювати пошук(поле з назвою "Пошук"),
 *здійснювати фільтрацію по списку(поле з назвою "Фільтрація").
Щоб додати новий запис, потрібно насамперед натиснути "Додати", потім ввести необхідні дані, додати обкладинку та настинути ОК.
Програма сповістить про успішність/неуспішність додавання нової книги.

Використовувати СУБД MS Access 2007+.
